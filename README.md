## HTTP server binding for Calcit

> Rust HTTP library for Calcit runtime.

### Usages

APIs:

```cirru
http.core/serve-http? $ {}
  :port 4000
```

Install to `~/.config/calcit/modules/`, compile and provide `*.{dylib,so}` file with `./build.sh`.
